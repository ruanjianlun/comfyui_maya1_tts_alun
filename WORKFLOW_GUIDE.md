# Maya1 TTS 工作流使用指南

## 基础工作流

### 工作流结构

```
[文本输入] -> [Maya1 TTS节点] -> [保存音频] / [预览音频]
```

### 节点配置说明

#### 1. Maya1 TTS 节点

**基本参数：**

| 参数 | 说明 | 推荐值 |
|-----|------|--------|
| text | 输入文本 | 50-200字符 |
| voice_preset | 语音风格 | 根据需求选择 |
| chunk_length | 分块长度 | 50（默认） |
| temperature | 随机性 | 0.3-0.5 |

**语音预设说明：**

- **男性-成熟**：适合叙事、教学、播报
- **女性-温柔**：适合客服、引导、温馨提示
- **男性-活力**：适合广告、促销、激励
- **女性-专业**：适合商务、新闻、正式场合
- **中性-播音**：适合标准播音、公告

#### 2. 保存音频节点

自动保存生成的音频到 ComfyUI 输出目录。

#### 3. 预览音频节点

在浏览器中直接预览播放音频。

## 高级工作流

### 多语音合成

使用多个 Maya1 TTS 节点，为不同角色生成不同的声音：

```
[文本A] -> [Maya1 TTS (男性)] -> [音频A]
[文本B] -> [Maya1 TTS (女性)] -> [音频B]
```

### 批量生成

使用循环节点批量处理多个文本：

```
[文本列表] -> [循环] -> [Maya1 TTS] -> [批量保存]
```

## 最佳实践

### 文本准备

1. **使用完整句子**：确保标点符号完整
2. **适当断句**：长文本建议手动分段
3. **英文文本**：当前版本对英文支持最好
4. **避免特殊字符**：清理掉不必要的符号

### 参数调优

**获得更自然的语音：**
- 调低 temperature (0.3-0.4)
- 使用较短的 chunk_length (30-50)

**获得更有变化的语音：**
- 调高 temperature (0.5-0.7)
- 使用适中的 chunk_length (50-70)

**加快生成速度：**
- 减小 chunk_length
- 使用较短的文本

### 质量优化

1. **检查文本**：确保语法正确
2. **选择合适预设**：根据内容选择匹配的声音
3. **分段处理**：长文本分多次生成
4. **多次尝试**：同样参数可能产生不同效果

## 示例工作流

项目包含 `workflow_example.json`，可直接导入使用：

1. 打开 ComfyUI
2. 点击 "Load" 按钮
3. 选择 `workflow_example.json`
4. 修改文本内容
5. 点击 "Queue Prompt" 开始生成

## 常见应用场景

### 1. 有声书制作

```yaml
参数配置：
  voice_preset: 男性-成熟 或 女性-专业
  chunk_length: 100-150
  temperature: 0.3-0.4
```

### 2. 视频配音

```yaml
参数配置：
  voice_preset: 根据角色选择
  chunk_length: 50-80
  temperature: 0.4-0.5
```

### 3. 语音提示

```yaml
参数配置：
  voice_preset: 女性-温柔 或 中性-播音
  chunk_length: 30-50
  temperature: 0.3
```

### 4. 广告配音

```yaml
参数配置：
  voice_preset: 男性-活力
  chunk_length: 40-60
  temperature: 0.5-0.6
```

## 技巧与窍门

1. **预热生成**：首次生成可能较慢，之后会快很多
2. **GPU 加速**：确保使用 GPU 可大幅提升速度
3. **批量处理**：相同设置下批量生成效率更高
4. **音频拼接**：使用音频编辑软件后期处理效果更好
